##simDHT:
A DHT crawler, is not quite based on Kademlia, written by Python, the code is very very simple.

##dependencies:
1. [bencode](https://pypi.python.org/pypi/bencode/1.0)

##start simDHT
```bash
python simDHT.py
```

##notice:
You can edit `max_node_qsize` value for speed.
